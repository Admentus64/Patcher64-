Patches for Bomberman Hero:
1. Virtual Console Exclusive patch options
	* Inject ROM (requires Z64 ROM)
	* Patch VC Emulator Only
	* Extract ROM Only
2. Free Patching (requires .IPS, .BPS, .XDELTA, .VCDIFF or .PPF File)
3. Patch Header
4. Additional Options

Known Issues:
- Unknown

Requirements:
- US Z64 ROM or US VC WAD for Bomberman Hero 64

Instructions:
- Select Z64 ROM or VC WAD
- Select patch
- Customize Virtual Console options
- Enable optional "Enable Options" checkbox
- Customize your patch within "Select Options"
- Press "Patch Selected Options"

Information:
- Original Z64 ROM or VC WAD is preserved
- Patches are mostly compatible with existing AR/Gecko Codes
    
Programs:
- Wad Packer/Wad Unpacker
- Wiimm's 'wszst' Tool
- Floating IPS
- RN64CRC
