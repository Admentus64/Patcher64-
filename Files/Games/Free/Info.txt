Patches for custom VC WAD:
1. ROM Injection (requires .Z64 ROM)
2. Free Patching (requires .IPS, .BPS, .XDELTA, .VCDIFF or .PPF File)

Known Issues:
- Unknown

Requirements:
- Any VC WAD File

Instructions:
- Select VC WAD File
- Enable optional VC Options checkboxes
- Press Inject ROM or Apply Patch
- Original WAD is preserved

Information:
- Existing AR/Gecko codes likely will not work

Programs:
- Wad Packer/Wad Unpacker
- Floating IPS
- Xdelta
- Xdelta3
- PPF3
- Wiimm's 'wszst' Tool
- RN64CRC
