Patch Options for Majora's Mask:
1. Virtual Console Exclusive patch options
	* Inject ROM (requires Z64 ROM)
	* Patch VC Emulator Only
	* Extract ROM Only
2. Free Patching (requires .IPS, .BPS, .XDELTA, .VCDIFF or .PPF File)
3. Patch Header
4. Three ROM hacks
5. Six fan translations
6. Redux and Additional Options

Known Issues:
- Unknown

Requirements:
- US Z64 ROM of US VC WAD for The Legend of Zelda: Majora's Mask

Instructions:
- Select Z64 ROM or VC WAD
- Select patch
- Customize Virtual Console options
- Enable optional "Enable Redux" or "Enable Options" checkboxes
- Customize your patch within "Select Options"
- Press "Patch Select Options"

Documentation / Sources:
- https://github.com/ShadowOne333/Zelda64-Redux-Documentation

Information:
- Original Z64 ROM or VC WAD is preserved
- Patches are mostly compatible with existing AR/Gecko Codes
- Redux forces "Remap D-Pad"
- "Expand Memory" renders AR/Gecko Codes unusable
