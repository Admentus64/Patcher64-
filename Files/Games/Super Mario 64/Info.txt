Patches Super Mario 64 game WAD:
1. ROM Injection (requires .Z64 ROM)
2. Free BPS Patching (requires .IPS or .BPS File)
3. Super Mario 64: 60 FPS v2 (Native 60 FPS support)
4. Super Mario 64: Free Cam (Analog Camera)
5. SM64: Multiplayer (V1.4.2)

Known Issues:
- Mario camera is inoperable (60 FPS)
- Intro demo is broken (60 FPS)

Requirements:
- Super Mario 64 VC USA WAD File

Instructions:
- Select Super Mario 64 VC USA WAD File
- Press one of several patching buttons
- Original WAD is preserved

Information:
- Existing AR/Gecko codes still work (60 FPS / Analog Camera)
- Existing AR/Gecko codes do not work (Multiplayer)
- Enable second emulated controller (Analog Camera / Multiplayer)
- Bind second emulated Control Stick to primary physical controller
