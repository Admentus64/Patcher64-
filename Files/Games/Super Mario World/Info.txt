Patch Options for Super Mario World:
1. Virtual Console Exclusive patch options
	* Inject ROM (requires SFC ROM)
	* Patch VC Emulator Only
	* Extract ROM Only
2. Apply Patch (requires a patch file)
3. Patch Header
4. Four ROM hacks

Known Issues:
- Unknown

Requirements:
- US SFC ROM or US VC WAD for Super Mario World

Instructions:
- Select SFC ROM or VC WAD
- Select patch
- Press "Patch Selected Option"

Information:
- Original SFC ROM or VC WAD is preserved
- The VC WAD forces "Downgrade" to Rev 0
    
Programs:
- Wad Packer/Wad Unpacker
- Wiimm's 'wszst' Tool
- Floating IPS
