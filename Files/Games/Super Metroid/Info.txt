Patch Options for Super Metroid:
1. Virtual Console Exclusive patch options
	* Inject ROM (requires SFC ROM)
	* Patch VC Emulator Only
	* Extract ROM Only
2. Apply Patch (requires a patch file)
3. Patch Header
4. Redux and Additional Options

Known Issues:
- Unknown

Requirements:
- US SFC ROM or US VC WAD for Super Metroid

Instructions:
- Select SFC ROM or VC WAD
- Select patch
- Enable optional "Enable Redux" or "Enable Options" checkboxes
- Customize your patch within "Select Options" and "Redux"
- Press "Patch Selected Option"

Information:
- Original SFC ROM or VC WAD is preserved
- Redux should be compatible with existing AR/Gecko Codes
    
Programs:
- Wad Packer/Wad Unpacker
- Wiimm's 'wszst' Tool
- Floating IPS
